# Social Media Best Time Poster

## Description
This Python desktop application analyzes social media engagement data and recommends the best time to post based on average likes.

## Technologies Used
- Python
- PyQt5
- Pandas
- Matplotlib

## Features
- Load CSV file
- Analyze engagement
- Recommend best posting time
- Display chart

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python main.py
